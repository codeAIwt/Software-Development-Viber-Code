## 测试环境准备

### 硬件环境要求
- 操作系统：Windows 10/11, macOS 10.15+, Ubuntu 18.04+
- 内存：至少4GB RAM
- 存储：至少2GB可用空间
- 网络：稳定的互联网连接
- 摄像头：用于AI检测功能测试（可选）

### 软件环境配置

#### 后端环境
```bash
# 1. 进入后端目录
cd backend

# 2. 安装Python依赖
pip install -r requirements.txt

# 3. 启动后端服务
uvicorn app:app --reload --host 0.0.0.0 --port 8000
```

#### 前端环境
```bash
# 1. 进入前端目录
cd frontend

# 2. 安装Node.js依赖
npm install

# 3. 启动前端开发服务器
npm run dev
```

#### 数据库配置
- **MySQL**：确保MySQL服务运行，数据库名为`online_study`
- **Redis**：确保Redis服务运行在默认端口6379

### 测试数据准备
```sql

TRUNCATE TABLE room_user;

```

## 测试用例设计

### 1. 用户注册登录测试

#### 测试用例ID：TC-001
**测试目标**：验证用户注册、登录、首次登录标签选择功能

**前置条件**：
- 后端服务正常运行（端口8000）
- 前端服务正常运行（端口5173）
- 数据库连接正常

**测试步骤**：
1. 访问首页 http://localhost:5173
2. 点击"注册"按钮
3. 输入测试数据：
   - 用户名：testuser001
   - 密码：Test123456
   - 昵称：测试用户001
4. 点击"注册"按钮
5. 注册成功后自动跳转到登录页面
6. 输入用户名密码：testuser001 / Test123456
7. 点击"登录"按钮
8. 首次登录时检查是否弹出标签选择弹窗
9. 选择3个标签（如：考研、英语、编程）
10. 点击"确认"按钮
11. 验证是否成功跳转到首页

**预期结果**：
- ✅ 注册成功，用户信息存入数据库
- ✅ 登录成功，生成JWT token
- ✅ 首次登录显示标签选择弹窗
- ✅ 标签选择后保存到用户信息
- ✅ 成功跳转到首页

**测试数据**：
