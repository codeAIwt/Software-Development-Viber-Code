# 第三阶段测试预期（数据库模块）

## 测试目标
验证数据库模块的功能是否正常，包括：
1. 数据库连接与初始化
2. 表结构创建
3. 数据存储与查询
4. 事务处理

## 测试环境
- 数据库：MySQL 8.0
- 后端：Python 3.12 + FastAPI
- 前端：Vue 3 + Vite

## 测试步骤

### 1. 数据库连接测试
- [ ] 启动后端服务，检查数据库连接是否成功
- [ ] 查看数据库表结构是否自动创建

### 2. 用户模块测试
- [ ] 注册新用户，检查数据库中是否生成用户记录
- [ ] 登录用户，检查数据库中用户信息是否正确
- [ ] 更新用户标签，检查数据库中标签是否更新

### 3. 自习室模块测试
- [ ] 创建自习室，检查数据库中是否生成自习室记录
- [ ] 加入自习室，检查数据库中是否生成房间成员记录
- [ ] 离开自习室，检查数据库中房间成员记录是否更新
- [ ] 销毁自习室，检查数据库中自习室记录是否更新

### 4. 学习时长模块测试
- [ ] 检查学习时长记录是否正确存储
- [ ] 检查学习时长统计是否正确

## 测试预期结果

### 1. 数据库连接
- 后端服务启动成功，无数据库连接错误
- 数据库表结构自动创建完成

### 2. 用户模块
- 注册成功后，数据库中存在新用户记录
- 登录成功后，数据库中用户信息正确
- 更新标签后，数据库中用户标签更新成功

### 3. 自习室模块
- 创建自习室后，数据库中存在自习室记录，且信息正确
- 加入自习室后，数据库中存在房间成员记录
- 离开自习室后，房间成员记录中的离开时间正确更新
- 销毁自习室后，自习室状态更新为closed

### 4. 学习时长模块
- 学习时长记录正确存储在数据库中
- 学习时长统计功能正常

## 测试注意事项
- 确保MySQL服务正常运行
- 确保数据库连接字符串正确
- 确保数据库用户权限足够
- 测试过程中注意观察控制台输出，确认数据操作是否正确

## 测试验证方法
- 使用MySQL客户端查看数据库表结构和数据
- 观察后端控制台输出的日志信息
- 检查前端页面显示的信息是否与数据库一致
